# Calendar

一个简单的原生前端日历项目，支持按年月查看日期、返回今天，并展示法定节假日信息。

## 功能

- 根据系统日期默认定位到当月
- 支持手动输入年份和月份跳转
- 支持按钮切换年份和月份
- 支持一键返回今天
- 高亮显示今天和当前选中日期
- 区分工作日、周末和空白占位格
- 展示节假日与调休信息
- 适配移动端显示

## 运行方式

这是一个静态网页项目，本地启用了静态服务器，可通过服务器访问项目 `index.html`。

## 项目结构

```text
calendar/
├─ index.html            # 页面入口
├─ scripts/              # 日历渲染与交互逻辑
├─ styles/               # 页面样式
└─ data/                 # 2000-2026 年节假日数据
```

## 数据说明

- `data/` 目录内提供了 `2000` 到 `2026` 年的节假日数据
- 超出本地数据范围的年份，会尝试从远程地址获取对应数据
