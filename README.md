# Calendar

一个可交互的网页日历。当前已支持根据系统日期初始化月份，动态生成当月日期，直接输入年份和月份跳转，点击“今天”返回当前日期，并高亮今天与选中的日期。日历会区分周末、空白占位格，并提供基础 hover 反馈。
